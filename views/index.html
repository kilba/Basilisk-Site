Basilisk

Basilisk is an open source rendering engine suited for 2D and 3D graphics. It is written in C with OpenGL. (Vulkan soon!) 

	
    git clone https://github.com/nusbog/bssh-client/
    cmake -S . -B build -G "MinGW Makefiles"
    cd ./build && make

It is recommended to add the executable to your environment variables. For further use, check out the manual.
