Basilisk

H1?

Basilisk is an open source rendering engine suited for 2D and 3D graphics. It is written in C, using OpenGL and GLFW.


Features

☑ - Object Rendering

☑ - Custom Framebuffers

☑ - Shaders (Vertex, Fragment, Geometry, Compute)

☑ - Model Loading

☐ - Text Rendering

☐ - Bindless Textures

☐ - Inverse Kinematics

☐ - Getting rid of GLFW, CGLM and CGLTF


Getting Started

Bssh

Bssh (Basilisk Shell) is an open source CLI used for updating the engine, downloading and uploading packages, this is the recommended way to use Basilisk.

Build Yourself

You can always compile the engine
